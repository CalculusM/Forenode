# Forenode

> **BIM·AI 민자사업 인텔리전스** — 민자도로 전 생애주기 수익성·리스크 분석 플랫폼

2026년 국토·교통 데이터 활용 경진대회 출품작 (제품/서비스 분야)

---

## 핵심 가치

**민자도로 1건 분석을 30초에 완료.** 신용평가사 6주 · 컨설팅 수천만원을 SaaS 월정액으로 대체.

```
신용평가사·자문사 6주       →  Forenode 30초
컨설팅 1건 수천만~수억원     →  SaaS 월 99만원~500만원
사전 검토만 분석             →  4시점 전 생애주기
6개 영역 분리 분석           →  단일 파이프라인 통합
```

---

## 5종 AI 모델 (가점 ④ AI 활용 만점)

| 모델 | 유형 | 알고리즘 | 검증 결과 |
|------|------|---------|----------|
| **XGBoost 수익성 등급** | 학습도구 | XGBoost + LOOCV | **93.2%** 정확도 |
| **Weibull 열화 모델** | 학습도구 | Weibull MLE | β=1.044, η=9.89년 |
| **OPEX 시계열** | 학습도구 | 다층 시계열 학습 | 4,380건 11년치 |
| **CAPEX 회귀** | 학습도구 | sklearn 회귀 | 화성-안성 **8%** 오차 |
| **RAG 법제 자문** | 분석도구 | ChromaDB + GPT-4o-mini | 16개 법령 1,963 청크 |

## 4기관 12종 공공 데이터

- **금융감독원 DART**: 13개 SPC × 5년 감사보고서
- **한국도로공사**: 포장보수 4,380건, 포장일반 54,760건, 안심구역 데이터
- **한국교통연구원 KOTI**: KTDB OD, 안심구역 결합 1 (PET 동형암호)
- **한국교통안전공단**: DTG 비식별 통계, 국토공간 가치지수 5종
- **국토안전관리원**: 교량 점검내역, 안심구역 결합 2 (PET 동형암호)
- **국가법령정보센터**: 16개 민자도로 관련 법령
- **한국은행 ECOS**: 기준금리·물가지수 API (실시간 연동)

## 핵심 기능

### 시점 탭 4개 (전 생애주기)

```
⏱ 사전 검토        → KDI PIMAC·주무관청 VfM 자동 판정
🏗 시공·자금조달    → 대주단(LTA)·사업주(STA) TDD 30초 (NEW!)

🔥 통행료 (도공 1.1배 적정성 영역)

📍 벤치마크 (13개 SPC)
📚 법제 RAG (16개 법령)
🎯 수익성 등급 (XGBoost)
🔧 Weibull (열화 분포)
💰 OPEX (시계열 학습)
```

## 차별점

### 1. 시점 × 모드 매트릭스 (국내외 부재)

```
            통계 모드 (±20%)    BIM 모드 (±5%, Stage 2)
사전 검토    ✅                 ─
시공·자금    ✅ LTA·STA·통합     Stage 2 IFC 통합
운영         ✅                 Stage 2 BIM 통합
재구조화     ✅ 해지시지급금     Stage 2 시설물 자산
```

### 2. PET 동형암호결합 — 국내 최초 민자도로 적용

정부의 2024년 안심구역 PET(Privacy Enhancing Technology) 신기술을 적극 활용.
- 안심구역 결합 1: 개별차량 이동경로 × 차량통행지표 (KOTI × KOTI)
- 안심구역 결합 2: 교량별 점검내역 × 구조물 영상분석 (국토안전관리원 × 도로공사)

### 3. 2024.10 정부 「민간투자 활성화 방안」 직접 반영

- 환매금지형 인프라펀드 + BTO 위험가중치 400→100%
- BTO+BTL 결합 사업 신설 → 사업유형 5번째로 즉시 지원
- 공사비 변동 위험 분담 특례 → 시점 2 분석 반영
- 사용료 적정성 도공 1.1배 → 통행료 탭에 영역 시각화

### 4. 5종 AI 모델 정량 검증

단순 검색이 아닌 모든 모델의 정량 검증 결과 제시 (LOOCV·MLE 신뢰구간·실측 검증).
